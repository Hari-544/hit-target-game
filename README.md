# 🎯 Hit the Target Game

A fun browser-based mini-game built using **HTML, CSS, and JavaScript**.
Players upload an image which becomes a **moving toy target**. The goal is to click the target as many times as possible before the timer runs out.

This project demonstrates **frontend development concepts**, DOM manipulation, event handling, animations, and audio effects.

---

# 🚀 Live Gameplay Concept

1. Upload an image of a person or object.
2. The image becomes a **bowling-pin style toy target**.
3. Press **Start Game**.
4. The target moves randomly on the screen.
5. Click the target to score points.
6. Each hit increases the score.
7. The game runs for **30 seconds**.
8. When time ends, the **final score is displayed**.

---

# ✨ Features

* 📸 Upload your own image as the game target
* 🎯 Random moving target gameplay
* ⏱ 30-second countdown timer
* 🏆 Real-time score tracking
* 🔊 Hit sound when target is clicked
* 🔊 Game over sound when time finishes
* 🍞 Toast notification for game result
* 🔒 Basic protection against copying or dragging elements
* 🎮 Simple and fun interactive gameplay

---

# 🛠 Technologies Used

* **HTML5** – Page structure
* **CSS3** – Styling and animations
* **JavaScript (Vanilla JS)** – Game logic and DOM manipulation

No frameworks or libraries are used. The entire game runs in the browser.

# ⚙️ How to Run the Game Locally

1. Download or clone the repository.

```
git clone https://github.com/YOUR_USERNAME/hit-target-game.git
```

2. Open the project folder.

3. Open **index.html** in your browser.

No installation or dependencies are required.

# 🎮 How to Play

1. Upload an image using the file upload button.
2. Click **Start Game**.
3. The image becomes a moving target.
4. Click the target to earn points.
5. Each hit increases the score.
6. Try to score as many points as possible before the timer reaches zero.
